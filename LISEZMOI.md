## ⚠️ Si Problème

**MongoDB pas démarré ?**
→ Cherchez "mongod" dans le Gestionnaire des tâches

**Port 8081 occupé ?**
→ Fermez toutes les fenêtres Java et relancez START-BACKEND.bat

**Frontend ne charge pas ?**
→ Dans PowerShell : `cd frontend; npm start`
